# Toyota Used Car Sales Analysis

Exploratory data analysis of used Toyota car sales with a focus on hybrid vehicles.

## Project Goals
- Clean and validate the dataset
- Explore trends in used car sales
- Calculate business metrics such as Hybrid Share of Sales
- Compare hybrid vs non-hybrid vehicles

## Key Findings
- Hybrids account for ~30% of used Toyota sales.
- Hybrid vehicles have lower tax and better fuel economy.
- Hybrids tend to have higher prices but lower mileage.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter / Google Colab
